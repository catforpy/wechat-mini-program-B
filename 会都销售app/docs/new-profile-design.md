# 新"我的"页面架构设计

## 📋 设计概述

新的"我的"页面采用**模块化可折叠**设计，将不同功能清晰地划分开来，用户可以按需展开查看。

## 🎯 页面结构

### 顶部用户信息卡片
- 显示用户头像、姓名、手机号
- 点击可登录/编辑个人信息

---

## 📦 五大功能模块

### 1. 我的注册（默认展开 ⬇️）

**功能说明**：管理公司/个体工商户资质

#### 展开内容：
- **资质列表**：
  - 企业资质（🏢）
  - 个体工商户资质（👤）
  - 显示统一社会信用代码
  - 显示认证状态（✓已认证 / 待审核）

- **添加新资质按钮**：点击可添加新的公司或个体工商户资质

#### 交互设计：
- 点击某个资质 → **进入旧版"我的"页面**（购买小程序、管理小程序）
- 不同公司资质对应不同的小程序资产

---

### 2. 我的合作（默认折叠 ▶）

**功能说明**：查看和管理合作小程序

#### 展开内容：
- **合作资质列表**：显示已合作的公司/组织
- **已参与合作的小程序**：显示通过合作方式获得的小程序
  - 显示小程序图标、名称
  - 显示合作角色（合作方）
  - 显示合作时间
- **操作按钮**：
  - 添加合作资质
  - 增加合作小程序

---

### 3. 我的租赁（默认折叠 ▶）

**功能说明**：查看和管理租赁小程序

#### 展开内容：
- **租赁资质列表**：显示租赁认证信息
- **已租赁的小程序**：显示租赁的小程序
  - 显示小程序图标、名称
  - 显示租赁期限（如：2024-01 ~ 2024-12）
  - 显示月租费用
- **操作按钮**：
  - 添加租赁资质
  - 增加租赁小程序

---

### 4. 我的关注（展开显示 ⭐）

**功能说明**：查看已关注的内容

#### 展开内容：
- 显示关注数量（右上角徽章）
- **关注列表**：
  - 小程序模板（如：餐饮点餐模板）
  - 小程序（如：唐极课得）
  - App
  - 每项显示图标、名称、类型
  - 每项有"取消关注"按钮
- 空状态：还没有关注任何内容

---

### 5. 我的钱包（展开显示 💰）

**功能说明**：管理资金账户

#### 展开内容：
- **余额卡片**：
  - 都达豆余额（左侧）
  - 人民币余额（右侧）
  - 渐变背景突出显示

- **兑换比例提示**：
  - 💡 兑换比例：10 都达豆 = 1 人民币

- **操作按钮**：
  - 💳 充值（绿色）
  - 💸 提现（橙色）
  - 📋 充值记录（蓝色）

## 🔄 页面跳转关系

```
新"我的"页面
│
├── 点击"我的注册"中的某个公司/个体工商户
│   └── → 跳转到 [公司详情页]（就是旧版"我的"页面）
│       ├── 我的小程序
│       ├── 模板市场

│       └── 小程序设置
│
├── 点击合作/租赁的小程序
│   └── → 跳转到 [小程序详情页]

├── 点击关注项

    └── → 跳转到 [充值记录页]
```

## 🎨 设计特点

### 1. 清晰的模块划分
- 每个模块有独立的图标和标题
- 折叠/展开状态一目了然
- 减少页面滚动，提升用户体验

### 2. 渐变色彩体系
- 主色调：紫色渐变（#667eea → #764ba2）
- 功能按钮使用不同颜色区分：
  - 充值：绿色（#e8f5e9）
  - 提现：橙色（#fff3e0）
  - 记录：蓝色（#e3f2fd）

### 3. 卡片式设计
- 所有内容块采用圆角卡片
- 阴影效果增加层次感
- 统一的内边距和外边距

### 4. 交互反馈
- 所有可点击项有active状态（透明度变化）
- 清晰的图标和文字说明
- 徽章显示数量（如关注数）

## 📊 数据结构

### 公司/个体工商户资质
```typescript
{
  id: number,
  name: string,                    // 公司/店铺名称
  type: 'company' | 'individual',  // 类型
  status: 'active' | 'pending',    // 认证状态
  unifiedSocialCreditCode: string, // 统一社会信用代码
  registrationDate: string         // 注册日期
}
```

### 合作小程序
```typescript
{
  id: number,
  name: string,        // 小程序名称
  emoji: string,       // 图标
  role: string,        // 合作角色
  cooperationDate: string // 合作时间
}
```

### 租赁小程序
```typescript
{
  id: number,
  name: string,        // 小程序名称
  emoji: string,       // 图标
  period: string,      // 租赁期限
  monthlyFee: string   // 月租费用
}
```

### 关注项
```typescript
{
  id: number,
  name: string,        // 名称
  type: string,        // 类型（小程序模板/小程序/App）
  emoji: string        // 图标
}
```

### 钱包余额
```typescript
{
  douDou: number,  // 都达豆
  rmb: number      // 人民币
}
```
