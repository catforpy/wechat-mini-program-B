# 交通服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 交通服务企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "交通服务"
- **成立时间**: date - 企业成立日期
- **服务规模**: string - 如: 覆盖xx个城市、车辆xx万辆

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **服务理念**: text - 服务宗旨、使命、愿景
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object
  - iOS链接: string
  - Android链接: string
  - 下载二维码: image

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

## 二、交通服务行业特性内容

### 2.1 服务范围
- **覆盖城市**: array
  - 城市名称: string
  - 开通时间: date
  - 运营状态: string
- **运营区域图**: image - 地图展示
- **服务网点**: array[10-50]
  - 网点名称: string
  - 网点地址: string
  - 服务时间: string
  - 联系电话: string
  - 网点图片: image

### 2.2 车辆信息
- **车辆概况**: object
  - 车辆总数: number
  - 车型分类: array
    - 车型: string
    - 数量: number
    - 车辆图片: image
- **车辆展示**: array[10-20]
  - 车型名称: string
  - 座位数: number
  - 车辆图片: array[3-5]
  - 车辆特点: text
- **车辆设施**: array
  - 设施名称: string (如: WiFi、充电口、空调)
  - 设施说明: text

### 2.3 服务产品
- **服务类型**: array
  - 服务名称: string (如: 快车、专车、拼车、包车)
  - 服务介绍: text
  - 服务特点: text
  - 价格说明: string
  - 服务图标: image
- **特色服务**: array[5-10]
  - 服务名称: string
  - 服务介绍: text
  - 服务图片: image

### 2.4 驾驶团队
- **司机概况**: object
  - 司机总数: number
  - 平均驾龄: number
  - 认证司机数: number
- **司机团队展示**: array[5-10]
  - 司机姓名: string
  - 驾龄: number
  - 服务单数: number
  - 好评率: number
  - 个人照片: image
  - 服务宣言: text

### 2.5 智能服务
- **APP功能**: array
  - 功能名称: string
  - 功能说明: text
  - 功能截图: image
- **特色功能**: array
  - 功能名称: string (如: 智能派单、实时定位、一键叫车)
  - 功能介绍: text
  - 功能视频: video

### 2.6 安全保障
- **安全措施**: array[5-10]
  - 措施名称: string (如: 身份认证、行程分享、紧急求助)
  - 措施说明: text
  - 措施图片: image
- **保险服务**: object
  - 保险种类: array
  - 理赔说明: text

### 2.7 价格体系
- **计费规则**: text
- **价格表**: array
  - 服务类型: string
  - 起步价: number
  - 里程费: number
  - 时长费: number
- **优惠活动**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string

### 2.8 数据统计
- **业务数据**: object
  - 累计服务人数: number
  - 累计订单数: number
  - 覆盖城市数: number
  - 安全行驶里程: number

### 2.9 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
