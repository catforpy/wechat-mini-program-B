# 体育行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 体育企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "体育"
- **成立时间**: date - 企业成立日期
- **服务规模**: string

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **服务理念**: text
- **发展历程**: array

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array

### 1.4 联系方式
- **联系电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、体育行业特性内容

### 2.1 体育项目
- **项目分类**: array
  - 类别名称: string (如: 球类、健身、游泳、瑜伽)
  - 项目列表: array
    - 项目名称: string
    - 项目介绍: text
    - 项目价格: string
    - 项目图片: image

### 2.2 场地设施
- **场地列表**: array[10-20]
  - 场地名称: string
  - 场地图片: array[3-5]
  - 场地介绍: text
  - 容纳人数: number
  - 预订链接: string

### 2.3 教练团队
- **教练列表**: array[5-10]
  - 教练姓名: string
  - 教练照片: image
  - 执教项目: string
  - 从业年限: number
  - 个人简介: text

### 2.4 培训课程
- **课程列表**: array[10-20]
  - 课程名称: string
  - 课程介绍: text
  - 课程时长: string
  - 课程价格: number
  - 开课时间: string

### 2.5 会员服务
- **会员类型**: array
  - 会员名称: string
  - 会员价格: number
  - 会员权益: array

### 2.6 数据统计
- **业务数据**: object
  - 场地数量: number
  - 教练人数: number
  - 会员数量: number
  - 日均接待: number

---

1. 企业介绍
