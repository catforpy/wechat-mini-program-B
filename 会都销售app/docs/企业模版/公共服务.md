# 公共服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **机构名称**: string
- **机构Logo**: image
- **行业类型**: string - "公共服务"
- **成立时间**: date

### 1.2 机构介绍
- **机构简介**: text
- **服务理念**: text

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]

### 1.4 联系方式
- **服务热线**: string
- **机构地址**: string
- **官方网站**: string

---

## 二、公共服务行业特性内容

### 2.1 服务事项
- **事项列表**: array
  - 事项名称: string
  - 办理流程: text
  - 所需材料: text
  - 办理时限: string

### 2.2 办事指南
- **指南列表**: array
  - 指南标题: string
  - 指南内容: text
  - 操作指南: array[image]

### 2.3 通知公告
- **公告列表**: array[10-20]
  - 公告标题: string
  - 发布时间: date
  - 公告内容: text

---
