# 公益行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **机构名称**: string - 公益机构全称
- **机构Logo**: image - 机构标志图片
- **行业类型**: string - "公益"
- **成立时间**: date - 机构成立日期
- **机构性质**: string - 如: 慈善组织、公益基金会

### 1.2 机构介绍
- **机构简介**: text - 200-500字机构介绍
- **公益理念**: text - 公益使命、愿景
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 机构宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **联系电话**: string
- **捐赠热线**: string
- **机构地址**: string
- **官方网站**: string
- **微信公众号**: image

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、公益行业特性内容

### 2.1 公益项目
- **项目分类**: array
  - 类别名称: string (如: 教育助学、医疗救助、扶贫济困、环境保护)
  - 项目列表: array
    - 项目名称: string
    - 项目封面: image
    - 项目介绍: text
    - 筹款目标: number
    - 已筹金额: number
    - 捐赠人数: number
    - 项目详情链接: string
- **热门项目**: array[10-20]
  - 项目信息同上
  - 热度指数: number

### 2.2 捐赠方式
- **捐赠渠道**: array
  - 渠道名称: string (如: 线上捐赠、银行转账、现场捐赠)
  - 渠道说明: text
  - 捐赠链接: string
- **捐赠证书**: object
  - 是否提供: boolean
  - 证书样式: image
  - 领取方式: text

### 2.3 志愿服务
- **志愿活动**: array[10-20]
  - 活动名称: string
  - 活动时间: date
  - 活动地点: string
  - 活动介绍: text
  - 招募人数: number
  - 已报名: number
  - 报名链接: string
- **志愿者招募**: object
  - 招募条件: text
  - 报名方式: text
  - 志愿者福利: text

### 2.4 公益动态
- **新闻资讯**: array[20-50]
  - 资讯标题: string
  - 发布时间: date
  - 资讯摘要: text
  - 详细内容: text
  - 相关图片: array[image]
- **项目进展**: array[10-20]
  - 项目名称: string
  - 进展标题: string
  - 进展内容: text
  - 进展图片: array[image]
  - 发布时间: date

### 2.5 透明公开
- **财务公示**: object
  - 收支情况: text
  - 财务报表: array[file]
  - 审计报告: array[file]
- **项目反馈**: array[10-20]
  - 项目名称: string
  - 反馈内容: text
  - 反馈图片: array[image]
  - 反馈时间: date

### 2.6 合作伙伴
- **合作机构**: array[20-50]
  - 机构名称: string
  - 机构Logo: image
  - 合作内容: string

### 2.7 数据统计
- **公益数据**: object
  - 累计筹款: number
  - 项目数量: number
  - 捐赠人数: number
  - 志愿者人数: number
