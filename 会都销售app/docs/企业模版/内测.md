# 内测行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 测试平台全称
- **企业Logo**: image
- **行业类型**: string - "内测"
- **成立时间**: date
- **平台规模**: string

### 1.2 企业介绍
- **企业简介**: text
- **平台理念**: text
- **发展历程**: array

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array

### 1.4 联系方式
- **联系邮箱**: string
- **反馈渠道**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、内测行业特性内容

### 2.1 内测项目
- **项目列表**: array[10-20]
  - 项目名称: string
  - 项目图标: image
  - 项目介绍: text
  - 测试类型: string
  - 测试人数: number
  - 项目详情链接: string

### 2.2 测试者社区
- **测试者列表**: array[10-20]
  - 测试者昵称: string
  - 测试者头像: image
  - 参与项目数: number
  - 贡献值: number

### 2.3 反馈系统
- **反馈列表**: array[10-20]
  - 反馈标题: string
  - 反馈内容: text
  - 反馈状态: string
  - 反馈时间: date

### 2.4 奖励机制
- **奖励类型**: array
  - 奖励名称: string
  - 奖励说明: text
  - 奖励图片: image

### 2.5 数据统计
- **平台数据**: object
  - 注册测试者数: number
  - 内测项目数: number
  - 反馈总数: number

---
