# 医疗服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 医院/诊所全称
- **企业Logo**: image - 机构标志图片
- **行业类型**: string - "医疗服务"
- **成立时间**: date - 机构成立日期
- **机构等级**: string - 如: 三甲、二甲、社区诊所

### 1.2 企业介绍
- **机构简介**: text - 200-500字机构介绍
- **服务理念**: text - 办院理念、使命、愿景
- **发展历程**: array - 历史里程碑列表
  - 时间: date
  - 事件: string
  - 图片: image (可选)

### 1.3 宣传内容
- **宣传片**: video - 医院宣传视频
- **宣传图片轮播**: array[5-10] - 医院环境图片
  - 图片: image
  - 标题: string
  - 描述: text
- **荣誉资质**: array - 资质证书图片
  - 证书名称: string
  - 证书图片: image
  - 获得时间: date

### 1.4 联系方式
- **咨询电话**: string - 咨询热线
- **急救电话**: string - 急救电话
- **机构地址**: string - 详细地址
- **营业时间**: string - 门诊时间
- **官方网站**: string - 网址链接
- **微信公众号**: image - 公众号二维码

### 1.5 用户评价
- **用户评分**: number - 1-5星评分
- **评价总数**: number
- **用户好评**: array[10-20] - 患者评价
  - 用户头像: image
  - 用户昵称: string
  - 评分: number
  - 评价内容: text
  - 评价时间: date
  - 配图: array[image] (可选)

---

## 二、医疗服务行业特性内容

### 2.1 医生团队
- **医师概况**: object
  - 医生总数: number
  - 主任医师人数: number
  - 副主任医师人数: number
  - 博士导师人数: number
- **专家团队**: array[10-30]
  - 医生姓名: string
  - 职称: string (如: 主任医师、教授、博士生导师)
  - 所属科室: string
  - 从业年限: number
  - 擅长领域: text
  - 个人简介: text
  - 个人照片: image
  - 学术成就: text (可选)
  - 出诊时间: string
  - 预约挂号链接: string
- **知名专家**: array[5-10]
  - 医生姓名: string
  - 职称: string
  - 擅长领域: text
  - 个人照片: image
  - 专家称号: string (如: 享受国务院特殊津贴专家)

### 2.2 科室介绍
- **科室分类**: array
  - 科室类别: string (如: 内科、外科、妇产科、儿科)
  - 科室列表: array
    - 科室名称: string
    - 科室简介: text
    - 特色诊疗: text
    - 科室医生: array
      - 医生姓名: string
      - 职称: string
      - 擅长领域: string
    - 科室环境图片: array[image]
- **重点科室**: array[5-10]
  - 科室名称: string
  - 科室级别: string (如: 国家重点专科、省级重点专科)
  - 科室介绍: text
  - 特色技术: array
    - 技术名称: string
    - 技术介绍: text
  - 科室环境: array[image]
  - 成功案例: array (可选)

### 2.3 医疗设备
- **设备概况**: object
  - 设备总数: number
  - 进口设备数: number
  - 设备总价值: string
- **先进设备**: array[10-20]
  - 设备名称: string
  - 设备型号: string
  - 设备产地: string
  - 设备介绍: text
  - 临床应用: text
  - 设备图片: image
  - 设备价格: string (可选)
- **特色技术**: array[5-10]
  - 技术名称: string
  - 技术介绍: text
  - 技术优势: text
  - 适用范围: text
  - 技术视频: video (可选)

### 2.4 就诊服务
- **预约挂号**: object
  - 预约方式: array
    - 方式: string (如: 微信预约、电话预约、现场预约)
    - 说明: text
  - 预约流程: text
  - 预约须知: text
  - 预约二维码: image (可选)
- **门诊安排**: object
  - 门诊时间: string
  - 专家门诊时间: string
  - 各科室排班: array
    - 科室: string
    - 星期: string
    - 上午医生: array[string]
    - 下午医生: array[string]
- **急诊服务**: object
  - 急诊时间: string (如: 24小时)
  - 急诊电话: string
  - 急诊流程: text
  - 急诊须知: text
- **住院服务**: object
  - 住院部介绍: text
  - 病房类型: array
    - 病房类型: string (如: 普通病房、双人病房、VIP病房)
    - 病房设施: text
    - 病房价格: string
    - 病房图片: image
  - 住院流程: text
  - 探视时间: string

### 2.5 特色服务
- **核心优势**: array[3-6]
  - 优势标题: string (如: 专家会诊、先进设备、温馨服务)
  - 优势描述: text
  - 优势图标: image
- **特色诊疗项目**: array[5-15]
  - 项目名称: string
  - 项目介绍: text
  - 适应症: text
  - 技术优势: text
  - 治疗效果: text
  - 案例展示: array (可选)
    - 患者情况: text
    - 治疗过程: text
    - 治疗效果: text
    - 对比图片: array[image]
- **健康管理**: array
  - 服务项目: string (如: 健康体检、慢病管理、家庭医生)
  - 服务说明: text
  - 服务流程: text
  - 服务图片: image (可选)
- **便民服务**: array
  - 服务项目: string (如: 免费WiFi、停车场、餐厅)
  - 服务说明: text

### 2.6 就诊指南
- **就诊流程**: object
  - 流程图: image
  - 流程说明: text
  - 注意事项: text
- **交通指南**: object
  - 详细地址: string
  - 交通方式: array
    - 方式: string (如: 公交、地铁、自驾)
    - 线路说明: text
  - 地图定位: image
- **就诊须知**: object
  - 必带证件: array[string]
  - 注意事项: text
  - 禁忌事项: text
- **医保信息**: object
  - 医保类型: array (如: 职工医保、居民医保、新农合)
  - 报销比例: text
  - 医保说明: text

### 2.7 患者案例
- **康复案例**: array[10-30]
  - 患者昵称: string
  - 就诊科室: string
  - 主治医生: string
  - 病情描述: text
  - 治疗方案: text
  - 康复情况: text
  - 治疗前后对比: array[image] (可选)
  - 康复时间: date
  - 患者评价: text
- **成功手术**: array[10-20]
  - 手术名称: string
  - 手术难度: string
  - 手术医生: string
  - 手术效果: text
  - 案例图片: array[image]
  - 手术视频: video (可选)

### 2.8 健康资讯
- **健康科普**: array[20-50]
  - 标题: string
  - 科普类别: string (如: 疾病预防、健康养生)
  - 发布时间: date
  - 内容摘要: text
  - 详细内容: text
  - 相关图片: array[image]
  - 文章链接: string
- **疾病百科**: array
  - 疾病名称: string
  - 疾病简介: text
  - 症状表现: text
  - 治疗方法: text
  - 预防措施: text
  - 相关科室: string
- **健康活动**: array[5-10]
  - 活动标题: string
  - 活动时间: date
  - 活动地点: string
  - 活动内容: text
  - 活动海报: image
  - 报名方式: string

### 2.9 药品服务
- **药房服务**: object
  - 药房介绍: text
  - 药品种类: text
  - 取药流程: text
  - 药房营业时间: string
- **在线购药**: object (如适用)
  - 是否支持: boolean
  - 购药方式: string
  - 配送说明: text

### 2.10 价格信息
- **收费标准**: text - 收费政策说明
- **价格公示**: array
  - 项目名称: string
  - 项目类别: string (如: 检查费、治疗费、药品费)
  - 价格: number
  - 医保报销: boolean
  - 价格说明: text
- **体检套餐**: array (如适用)
  - 套餐名称: string
  - 套餐内容: text
  - 原价: number
  - 优惠价: number
  - 套餐详情: text

### 2.11 常见问题
- **FAQ**: array[10-20]
  - 问题: string
  - 答案: text
  - 展开图标: image (可选)

## 三、页面结构建议

### 3.1 页面顶部
- 轮播宣传图/视频
- 医院Logo + 名称
- 核心数据展示（如: 建院xx年、医生xx人、科室xx个）

### 3.2 快捷入口
- 预约挂号
- 在线咨询
- 报告查询
- 就诊指南

### 3.3 内容模块（按顺序）
1. 医院介绍 + 服务理念
2. 医生团队（专家介绍）
3. 科室介绍
4. 医疗设备
5. 特色诊疗项目
6. 就诊服务（预约、门诊、急诊）
7. 就诊指南（流程、交通、须知）
8. 患者案例
9. 健康资讯
10. 价格信息
11. 用户评价
12. 常见问题
13. 联系方式

### 3.4 底部
- 关注公众号
- 在线咨询
- 咨询电话
- 医院地址
