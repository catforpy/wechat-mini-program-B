# 咨询行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 咨询企业全称
- **企业Logo**: image
- **行业类型**: string - "咨询"
- **成立时间**: date
- **服务规模**: string

### 1.2 企业介绍
- **企业简介**: text
- **咨询理念**: text
- **发展历程**: array

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array

### 1.4 联系方式
- **咨询电话**: string
- **商务合作**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、咨询行业特性内容

### 2.1 咨询服务
- **服务分类**: array
  - 类别名称: string (如: 管理咨询、财务咨询、法律咨询)
  - 服务列表: array
    - 服务名称: string
    - 服务介绍: text
    - 服务流程: text
    - 服务价格: string

### 2.2 专家团队
- **专家列表**: array[5-15]
  - 专家姓名: string
  - 专家照片: image
  - 专业领域: string
  - 从业年限: number
  - 个人简介: text
  - 咨询方式: string

### 2.3 成功案例
- **案例列表**: array[10-20]
  - 案例标题: string
  - 客户名称: string
  - 咨询内容: text
  - 解决方案: text
  - 案例图片: array[image]

### 2.4 咨询流程
- **流程说明**: object
  - 流程图: image
  - 流程说明: text

### 2.5 数据统计
- **业务数据**: object
  - 服务企业数: number
  - 专家人数: number
  - 完成项目数: number

---

## 三、页面结构建议

1. 企业介绍

3. 专家团队
4. 成功案例
5. 咨询流程
6. 数据统计
7. 常见问题
