# 商业服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 商业服务企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "商业服务"
- **成立时间**: date - 企业成立日期
- **服务规模**: string - 如: 服务企业xx万家

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **服务理念**: text - 服务理念、使命
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **商务合作**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、商业服务行业特性内容

### 2.1 服务项目
- **服务分类**: array
  - 类别名称: string (如: 企业服务、营销服务、技术服务)
  - 服务列表: array
    - 服务名称: string
    - 服务介绍: text
    - 服务流程: text
    - 服务价格: string
    - 服务图片: image
    - 服务详情链接: string
- **热门服务**: array[10-20]
  - 服务信息同上
  - 订阅量: number
  - 好评率: number

### 2.2 行业解决方案
- **解决方案**: array[10-20]
  - 方案名称: string
  - 适用行业: string
  - 方案介绍: text
  - 方案图片: image
  - 成功案例: string
  - 方案详情链接: string

### 2.3 客户案例
- **成功案例**: array[10-30]
  - 客户名称: string
  - 客户Logo: image
  - 服务项目: string
  - 项目介绍: text
  - 项目成果: text
  - 案例图片: array[image]

### 2.4 专业团队
- **团队概况**: object
  - 团队规模: number
  - 专业人数: number
  - 平均经验: number
- **专家团队**: array[5-10]
  - 专家姓名: string
  - 职位: string
  - 专业领域: string
  - 从业年限: number
  - 个人照片: image
  - 个人简介: text

### 2.5 服务保障
- **核心优势**: array[3-6]
  - 优势标题: string
  - 优势描述: text
  - 优势图标: image
- **服务承诺**: array
  - 承诺项目: string
  - 承诺说明: text

### 2.6 数据统计
- **业务数据**: object
  - 服务企业数: number
  - 项目完成数: number
  - 客户满意度: number
  - 服务年限: number
