# 商家自营行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 商家全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "商家自营"
- **成立时间**: date - 企业成立日期
- **经营范围**: string - 主营业务范围

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **经营理念**: text - 经营理念、使命
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10] - 产品/门店图片
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、商家自营行业特性内容

### 2.1 产品展示
- **产品分类**: array
  - 类别名称: string
  - 产品列表: array
    - 产品名称: string
    - 产品图片: image
    - 产品介绍: text
    - 产品价格: number
    - 库存: number
    - 产品详情链接: string
- **热销产品**: array[10-20]
  - 产品信息同上
  - 销量: number
  - 好评率: number

### 2.2 门店信息
- **门店列表**: array[10-50]
  - 门店名称: string
  - 门店图片: array[3-5]
  - 门店地址: string
  - 营业时间: string
  - 联系电话: string
  - 门店详情链接: string

### 2.3 品牌故事
- **品牌介绍**: text
- **品牌理念**: text
- **品牌文化**: text
- **创始人**: object
  - 创始人姓名: string
  - 创始人照片: image
  - 创始人介绍: text

### 2.4 品质保障
- **核心优势**: array[3-6]
  - 优势标题: string
  - 优势描述: text
  - 优势图标: image
- **质量认证**: array
  - 认证名称: string
  - 认证图片: image

### 2.5 优惠活动
- **活动列表**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image

### 2.6 数据统计
- **业务数据**: object
  - 门店数量: number
  - 产品数量: number
  - 服务客户数: number
  - 年营业额: number

---

## 三、页面结构建议

1. 企业介绍
2. 品牌故事
3. 产品展示
4. 门店信息
5. 品质保障
6. 优惠活动
7. 数据统计
8. 用户评价
9. 联系方式
