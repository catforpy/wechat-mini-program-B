# 快递业与邮政行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string
- **企业Logo**: image
- **行业类型**: string - "快递业与邮政"
- **成立时间**: date
- **服务规模**: string

### 1.2 企业介绍
- **企业简介**: text
- **服务理念**: text
- **发展历程**: array

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、快递业与邮政行业特性内容

### 2.1 服务范围
- **覆盖区域**: array
  - 区域名称: string
  - 服务类型: array

### 2.2 服务产品
- **产品列表**: array
  - 产品名称: string (如: 标准快递、特快专递、经济快递)
  - 产品介绍: text
  - 时效说明: string
  - 价格说明: string

### 2.3 网点信息
- **网点列表**: array[10-50]
  - 网点名称: string
  - 网点地址: string
  - 营业时间: string
  - 联系电话: string

### 2.4 服务保障
- **保障措施**: array
  - 措施名称: string
  - 措施说明: text

### 2.5 数据统计
- **业务数据**: object
  - 网点数量: number
  - 日均处理量: number
  - 服务客户数: number

---

## 三、页面结构建议

1. 企业介绍
2. 服务范围
3. 服务产品
4. 网点信息
5. 服务保障
6. 数据统计
7. 联系方式
