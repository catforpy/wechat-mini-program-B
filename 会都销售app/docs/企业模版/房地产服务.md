# 房地产服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 房地产企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "房地产服务"
- **成立时间**: date - 企业成立日期
- **企业规模**: string - 如: 全国xx家分公司

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **服务理念**: text - 服务宗旨、使命、愿景
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10] - 楼盘/门店环境图片
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **咨询电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

## 二、房地产服务行业特性内容

### 2.1 在售楼盘
- **楼盘列表**: array[10-50]
  - 楼盘名称: string
  - 楼盘封面: image
  - 楼盘位置: string
  - 房价范围: string
  - 户型: array (如: 1室1厅、2室2厅)
  - 面积范围: string
  - 楼盘简介: text
  - 楼盘详情链接: string
- **热门楼盘**: array[5-10]
  - 楼盘名称: string
  - 楼盘封面: image
  - 价格: string
  - 浏览次数: number
  - 咨询次数: number

### 2.2 房源信息
- **房源分类**: array
  - 类别: string (如: 二手房、租房、商铺、写字楼)
  - 房源列表: array
    - 房源标题: string
    - 房源图片: image
    - 房源位置: string
    - 房源价格: string
    - 户型: string
    - 面积: string
    - 房源详情链接: string
- **最新房源**: array[10-20]
  - 房源信息同上
  - 发布时间: date

### 2.3 服务团队
- **经纪人概况**: object
  - 经纪人总数: number
  - 金牌经纪人: number
  - 平均从业年限: number
- **经纪人团队**: array[10-20]
  - 经纪人姓名: string
  - 从业年限: number
  - 成交单数: number
  - 好评率: number
  - 个人照片: image
  - 擅长区域: string
  - 个人简介: text
  - 联系方式: string

### 2.4 服务保障
- **核心优势**: array[3-6]
  - 优势标题: string (如: 真实房源、专业服务、透明交易)
  - 优势描述: text
  - 优势图标: image
- **服务承诺**: array
  - 承诺项目: string
  - 承诺说明: text
  - 承诺图片: image
- **产权保障**: object
  - 产权核实: text
  - 资金监管: text
  - 保障说明: text

### 2.5 楼盘详情（模版）
- **楼盘基本信息**: object
  - 楼盘名称: string
  - 开发商: string
  - 楼盘位置: string
  - 物业类型: string (如: 住宅、商铺、写字楼)
  - 建筑面积: string
  - 绿化率: string
  - 容积率: string
  - 车位数量: number
- **楼盘图片**: array[20-50]
  - 图片类型: string (如: 效果图、样板间、周边环境)
  - 图片: image
  - 图片说明: string
- **户型展示**: array[5-10]
  - 户型图: image
  - 户型名称: string (如: 3室2厅2卫)
  - 建筑面积: string
  - 套内面积: string
  - 户型介绍: text
- **周边配套**: array
  - 配套类型: string (如: 学校、医院、商场、地铁)
  - 配套名称: string
  - 配套距离: string
  - 配套图片: image
- **价格信息**: object
  - 参考价格: string
  - 价格详情: text
  - 优惠政策: text
- **销售信息**: object
  - 销售状态: string (如: 在售、售罄、待售)
  - 开盘时间: date
  - 交房时间: date
  - 售楼处地址: string
  - 售楼处电话: string

### 2.6 租房服务
- **租房列表**: array[20-50]
  - 房源标题: string
  - 房源图片: image
  - 房源位置: string
  - 租金: string
  - 户型: string
  - 面积: string
  - 配套设施: array (如: 冰箱、洗衣机、空调)
  - 房源详情链接: string
- **租房保障**: array
  - 保障项目: string (如: 真实房源、押金保障)
  - 保障说明: text

### 2.7 数据统计
- **业务数据**: object
  - 服务客户数: number
  - 成交房源数: number
  - 覆盖小区数: number
  - 经纪人数: number

### 2.8 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
