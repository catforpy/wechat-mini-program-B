# 政务民生行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **机构名称**: string - 政府机构/服务平台全称
- **机构Logo**: image - 机构标志图片
- **行业类型**: string - "政务民生"
- **成立时间**: date - 机构成立日期
- **机构性质**: string - 如: 政府部门、事业单位、公共服务平台

### 1.2 机构介绍
- **机构简介**: text - 200-500字机构介绍
- **服务理念**: text - 服务宗旨、使命、愿景
- **发展历程**: array - 历史里程碑列表
  - 时间: date
  - 事件: string
  - 图片: image (可选)

### 1.3 宣传内容
- **宣传片**: video - 机构宣传视频
- **宣传图片轮播**: array[5-10] - 机构环境/服务场景图片
  - 图片: image
  - 标题: string
  - 描述: text
- **荣誉资质**: array - 资质证书图片
  - 证书名称: string
  - 证书图片: image
  - 获得时间: date

### 1.4 联系方式
- **咨询电话**: string - 咨询热线
- **监督电话**: string - 监督投诉电话
- **机构地址**: string - 详细地址
- **办公时间**: string - 办公时间说明
- **官方网站**: string - 网址链接
- **微信公众号**: image - 公众号二维码

### 1.5 用户评价
- **用户评分**: number - 1-5星评分
- **评价总数**: number
- **用户好评**: array[10-20] - 群众评价
  - 用户头像: image
  - 用户昵称: string
  - 评分: number
  - 评价内容: text
  - 评价时间: date

---

## 二、政务民生行业特性内容

### 2.1 服务项目
- **服务分类**: array
  - 服务类别: string (如: 社保服务、公积金、户籍办理、证件办理)
  - 服务列表: array
    - 服务名称: string
    - 服务简介: text
    - 办理条件: text
    - 办理材料: text
    - 办理流程: text
    - 办理时限: string
    - 收费标准: string
    - 服务图标: image
    - 在线办理链接: string
- **热门服务**: array[10-20]
  - 服务名称: string
  - 服务简介: text
  - 办理人数: number
  - 好评率: number
  - 服务图标: image
  - 在线办理链接: string
- **最新上线**: array[5-10]
  - 服务名称: string
  - 上线时间: date
  - 服务介绍: text
  - 服务图片: image

### 2.2 办事指南
- **办事流程**: object
  - 流程图: image
  - 流程说明: text
  - 注意事项: text
- **办事大厅**: object
  - 大厅地址: string
  - 大厅照片: array[5-10]
  - 窗口设置: array
    - 窗口编号: string
    - 窗口名称: string
    - 办理业务: array[string]
  - 办公时间: string
  - 交通指南: text
- **网上办事**: object
  - 办事平台名称: string
  - 平台链接: string
  - 平台二维码: image
  - 使用教程: video (可选)
  - 功能介绍: array
    - 功能名称: string
    - 功能说明: text

### 2.3 政策解读
- **最新政策**: array[10-20]
  - 政策标题: string
  - 发布时间: date
  - 发布部门: string
  - 政策摘要: text
  - 详细内容: text
  - 政策文件: file (PDF)
  - 解读文章: text
- **政策分类**: array
  - 分类名称: string (如: 社保政策、住房政策、教育政策)
  - 政策列表: array
    - 政策标题: string
    - 发布时间: date
    - 政策链接: string
- **政策问答**: array[10-20]
  - 问题: string
  - 答案: text
  - 相关政策: string

### 2.4 通知公告
- **最新公告**: array[10-20]
  - 公告标题: string
  - 公告类型: string (如: 系统维护、放假通知、业务调整)
  - 发布时间: date
  - 公告内容: text
  - 相关图片: array[image]
- **重要通知**: array[5-10]
  - 通知标题: string
  - 通知级别: string (如: 紧急、重要)
  - 发布时间: date
  - 通知内容: text
  - 通知链接: string

### 2.5 特色服务
- **核心优势**: array[3-6]
  - 优势标题: string (如: 一站式服务、24小时在线)
  - 优势描述: text
  - 优势图标: image
- **便民服务**: array[5-15]
  - 服务名称: string (如: 预约办理、进度查询、证照快递)
  - 服务说明: text
  - 使用方式: text
  - 服务图片: image (可选)
- **智能服务**: array
  - 服务名称: string (如: 人脸识别、智能客服)
  - 服务介绍: text
  - 使用方法: text

### 2.6 数据统计
- **服务数据**: object
  - 服务总人数: number
  - 办理业务总数: number
  - 在线办理率: number
  - 平均办理时间: string
- **实时数据**: array
  - 数据名称: string
  - 数据值: number
  - 更新时间: date
- **数据图表**: array
  - 图表标题: string
  - 图表类型: string (如: 折线图、柱状图)
  - 图表数据: array
  - 图表图片: image

### 2.7 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
  - 相关链接: string (可选)
  - 展开图标: image (可选)

### 2.8 办事案例
- **成功案例**: array[10-20]
  - 案例标题: string
  - 办理业务: string
  - 办理人: string (匿名)
  - 办理时间: date
  - 办理过程: text
  - 办理结果: text
  - 用户评价: text
- **典型经验**: array[5-10]
  - 经验标题: string
  - 经验描述: text
  - 相关图片: array[image]

## 三、页面结构建议

### 3.1 页面顶部
- 轮播宣传图/视频
- 机构Logo + 名称
- 核心数据展示（如: 服务xx万人、办理xx万件）

### 3.2 快捷入口
- 热门服务
- 在线办理
- 进度查询
- 咨询反馈

### 3.3 内容模块（按顺序）
1. 机构介绍 + 服务理念
2. 服务项目（分类展示）
3. 热门服务
4. 办事指南
5. 政策解读
6. 通知公告
7. 特色服务
8. 数据统计
9. 办事案例
10. 常见问题
11. 用户评价
12. 联系方式

### 3.4 底部
- 关注公众号
- 在线咨询
- 咨询电话
- 机构地址
