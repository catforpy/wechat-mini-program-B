# 教育服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 企业/公司全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "教育服务"
- **成立时间**: date - 企业成立日期
- **企业规模**: string - 师生人数/校区数量

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **教育理念**: text - 办学理念、使命、愿景
- **发展历程**: array - 历史里程碑列表
  - 时间: date
  - 事件: string
  - 图片: image (可选)

### 1.3 宣传内容
- **宣传片**: video - 学校/机构宣传视频
- **宣传图片轮播**: array[5-10] - 校园风采图片
  - 图片: image
  - 标题: string
  - 描述: text
- **荣誉资质**: array - 资质证书图片
  - 证书名称: string
  - 证书图片: image
  - 获得时间: date

### 1.4 联系方式
- **咨询电话**: string - 咨询热线
- **招生电话**: string - 招生办电话
- **企业地址**: string - 详细地址
- **营业时间**: string - 营业时间说明
- **官方网站**: string - 网址链接
- **微信公众号**: image - 公众号二维码

### 1.5 用户评价
- **用户评分**: number - 1-5星评分
- **评价总数**: number
- **用户好评**: array[10-20] - 学员/家长评价
  - 用户头像: image
  - 用户昵称: string
  - 评分: number
  - 评价内容: text
  - 评价时间: date
  - 配图: array[image] (可选)

---

## 二、教育服务行业特性内容

### 2.1 师资力量
- **师资概况**: object
  - 教师总数: number
  - 高级教师人数: number
  - 外教人数: number (如有)
  - 平均教龄: number
- **名师团队**: array[5-15]
  - 教师姓名: string
  - 职称: string (如: 高级教师、特级教师、教授)
  - 教授科目: string
  - 从业年限: number
  - 教学成果: text
  - 个人简介: text
  - 个人照片: image
  - 荣誉证书: array[image] (可选)
  - 教学视频: video (可选)
- **外教团队**: array[5-10] (如有)
  - 外教姓名: string
  - 国籍: string
  - 教授科目: string
  - 教学特色: text
  - 个人照片: image
  - 教学视频: video (可选)
- **教师资质认证**: array
  - 认证名称: string
  - 认证图片: image
  - 获得时间: date

### 2.2 课程体系
- **课程分类**: array
  - 课程类别: string (如: 学科辅导、兴趣培养、职业技能)
  - 课程列表: array
    - 课程名称: string
    - 课程简介: text
    - 适合人群: string
    - 课程时长: string
    - 课程价格: number
    - 课程封面: image
    - 课程详情页链接: string
- **热门课程**: array[5-10]
  - 课程名称: string
  - 课程封面: image
  - 课程介绍: text
  - 课时数: number
  - 价格: number
  - 报名人数: number
  - 好评率: number
  - 课程试听: video (可选)
- **课程表**: array (如适用)
  - 时间段: string
  - 星期: string
  - 课程名称: string
  - 授课教师: string
  - 教室: string

### 2.3 学员风采
- **学员展示**: array[10-30]
  - 学员姓名: string (或昵称)
  - 学习课程: string
  - 学习时长: string
  - 学习成果: text
  - 学员照片: image
  - 作品展示: array[image] (可选)
  - 获奖情况: string (可选)
- **优秀毕业生**: array[5-15]
  - 姓名: string
  - 毕业时间: date
  - 录取学校/工作单位: string
  - 个人照片: image
  - 成长感言: text
- **学员成绩展示**: array
  - 学员姓名: string
  - 考试类别: string
  - 成绩/分数: string
  - 排名: string (可选)
  - 成绩单: image (可选)

### 2.4 教学环境
- **校区环境**: object
  - 校区数量: number
  - 总面积: string
  - 校区图片: array[20-30]
    - 图片类型: string (如: 教室、图书馆、实验室、运动场)
    - 图片: image
    - 图片说明: string
- **教学设施**: array[5-10]
  - 设施名称: string (如: 多媒体教室、科学实验室、美术室)
  - 设施介绍: text
  - 设施图片: array[3-5]
  - 设施数量: number (可选)
- **安全措施**: array
  - 安全项目: string (如: 24小时安保、监控全覆盖)
  - 安全说明: text
  - 安全证书: image (可选)

### 2.5 教学特色
- **核心优势**: array[3-6]
  - 优势标题: string (如: 小班教学、一对一辅导、个性化定制)
  - 优势描述: text
  - 优势图标: image
- **教学方法**: object
  - 教学模式: string (如: 线下教学、线上教学、混合教学)
  - 教学特色说明: text
  - 教学流程图: image (可选)
- **个性化服务**: array
  - 服务项目: string (如: 学习诊断、学习计划制定、作业辅导)
  - 服务说明: text
  - 服务图片: image (可选)

### 2.6 招生信息
- **招生对象**: text - 招生年龄/年级说明
- **招生计划**: object
  - 招生人数: number
  - 班级设置: array
    - 班型: string (如: 小班、中班、大班、一对一)
    - 班级人数: number
    - 上课时间: string
- **报名时间**: string - 报名起止时间
- **报名方式**: array
  - 方式: string (如: 线上报名、电话报名、现场报名)
  - 说明: text
- **入学测试**: object (如适用)
  - 是否需要测试: boolean
  - 测试内容: text
  - 测试时间: string
  - 测试地点: string

### 2.7 收费标准
- **收费说明**: text - 收费政策说明
- **价格表**: array
  - 课程类别: string
  - 课程名称: string
  - 课时数: number
  - 原价: number
  - 优惠价: number
  - 价格说明: text
- **优惠政策**: array
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image
- **退费政策**: text - 退费规则说明

### 2.8 在线服务
- **在线课程平台**: object (如适用)
  - 平台名称: string
  - 平台链接: string
  - 二维码: image
  - 功能介绍: array
- **预约试听**: object
  - 是否支持: boolean
  - 预约方式: string
  - 试听课程列表: array
    - 课程名称: string
    - 试听时间: string
    - 剩余名额: number
- **学习管理系统**: object (如适用)
  - 系统介绍: text
  - 功能列表: array
  - 系统截图: array[image]

### 2.9 教学成果
- **升学数据**: object (如适用)
  - 升学率: number
  - 重点学校录取率: number
  - 历年录取情况: array
    - 年份: number
    - 录取人数: number
    - 录取学校: array[string]
- **竞赛获奖**: array
  - 获奖时间: date
  - 竞赛名称: string
  - 获奖学员: array[string]
  - 奖项级别: string
  - 奖状图片: image
- **教学成果展示**: array[10-20]
  - 标题: string
  - 成果描述: text
  - 成果图片: array[image]
  - 相关视频: video (可选)

### 2.10 活动资讯
- **最新活动**: array[5-10]
  - 活动标题: string
  - 活动时间: date
  - 活动地点: string
  - 活动内容: text
  - 活动海报: image
  - 报名方式: string
- **新闻资讯**: array[10-20]
  - 标题: string
  - 发布时间: date
  - 内容摘要: text
  - 详细内容: text
  - 相关图片: array[image]
  - 文章链接: string

### 2.11 常见问题
- **FAQ**: array[10-20]
  - 问题: string
  - 答案: text
  - 展开图标: image (可选)

## 三、页面结构建议

### 3.1 页面顶部
- 轮播宣传图/视频
- 企业Logo + 名称
- 核心数据展示（如: 建校xx年、教师xx人、学员xx万人）

### 3.2 快捷入口
- 预约试听
- 在线咨询
- 课程报名
- 找校区

### 3.3 内容模块（按顺序）
1. 企业介绍 + 教育理念
2. 师资力量（名师团队）
3. 课程体系
4. 教学环境
5. 教学特色
6. 学员风采 + 优秀毕业生
7. 教学成果
8. 招生信息
9. 收费标准
10. 在线服务
11. 活动资讯
12. 用户评价
13. 常见问题
14. 联系方式

### 3.4 底部
- 关注公众号
- 在线咨询
- 咨询电话
- 校区地址
