# 文娱行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 文娱企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "文娱"
- **成立时间**: date - 企业成立日期
- **服务规模**: string - 如: 用户xx亿、内容xx万

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **品牌理念**: text - 品牌文化、理念
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、文娱行业特性内容

### 2.1 内容分类
- **内容类别**: array
  - 类别名称: string (如: 音乐、视频、直播、小说、漫画)
  - 内容列表: array
    - 内容标题: string
    - 内容封面: image
    - 内容介绍: text
    - 播放量/阅读量: number
    - 点赞数: number
    - 内容详情链接: string
- **热门内容**: array[10-20]
  - 内容信息同上
  - 热度指数: number

### 2.2 歌曲/专辑
- **歌曲列表**: array[20-50]
  - 歌曲名称: string
  - 歌手: string
  - 专辑: string
  - 时长: string
  - 播放量: number
  - 歌曲链接: string
- **专辑列表**: array[10-20]
  - 专辑名称: string
  - 歌手: string
  - 专辑封面: image
  - 发行时间: date
  - 歌曲数量: number
  - 专辑介绍: text
  - 专辑详情链接: string

### 2.3 视频/影视
- **视频分类**: array
  - 类别: string (如: 电影、电视剧、综艺、动漫)
  - 视频列表: array
    - 视频标题: string
    - 视频封面: image
    - 导演/演员: string
    - 时长: string
    - 播放量: number
    - 视频详情链接: string
- **热门视频**: array[10-20]
  - 视频信息同上
  - 评分: number

### 2.4 直播
- **直播列表**: array[10-20]
  - 主播昵称: string
  - 直播封面: image
  - 直播标题: string
  - 直播间分类: string
  - 观看人数: number
  - 直播状态: string
  - 直播链接: string
- **主播推荐**: array[5-10]
  - 主播信息同上
  - 推荐理由: text

### 2.5 原创内容
- **原创作品**: array[10-20]
  - 作品标题: string
  - 作者: string
  - 作品封面: image
  - 作品介绍: text
  - 作品链接: string
- **UP主/创作者**: array[10-20]
  - 创作者昵称: string
  - 创作者头像: image
  - 粉丝数: number
  - 作品数: number
  - 个人简介: text
  - 创作者主页链接: string

### 2.6 排行榜
- **排行榜列表**: array
  - 榜单名称: string (如: 热歌榜、新歌榜、飙升榜)
  - 榜单周期: string (如: 日榜、周榜、月榜)
  - 榜单内容: array
    - 排名: number
    - 内容信息 (同上)
- **榜单详情**: object
  - 榜单名称: string
  - 更新时间: date
  - 榜单列表: array

### 2.7 会员服务
- **会员类型**: array
  - 会员名称: string (如: 普通会员、VIP会员、SVIP会员)
  - 会员价格: number
  - 会员时长: string
  - 会员权益: array
    - 权益名称: string
    - 权益说明: text
- **优惠活动**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image

### 2.8 数据统计
- **平台数据**: object
  - 注册用户数: number
  - 日活跃用户: number
  - 内容总数: number
  - 总播放量: number

### 2.9 社区互动
- **用户评论**: array[10-20]
  - 用户昵称: string
  - 评论内容: text
  - 评论时间: date
  - 点赞数: number
- **话题讨论**: array[10-20]
  - 话题标题: string
  - 话题热度: number
  - 参与人数: number
  - 话题链接: string
