# 旅游服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 旅游服务企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "旅游服务"
- **成立时间**: date - 企业成立日期
- **服务规模**: string - 如: 覆盖xx个国家、xx条线路

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **服务理念**: text - 服务宗旨、使命
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10] - 旅游目的地图片
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、旅游服务行业特性内容

### 2.1 旅游线路
- **线路分类**: array
  - 类别名称: string (如: 国内游、出境游、周边游、定制游)
  - 线路列表: array
    - 线路名称: string
    - 线路封面: image
    - 目的地: string
    - 行程天数: number
    - 价格: number
    - 出团日期: array[date]
    - 线路介绍: text
    - 线路详情链接: string
- **热门线路**: array[10-20]
  - 线路信息同上
  - 销量: number
  - 好评率: number

### 2.2 目的地推荐
- **目的地列表**: array[10-50]
  - 目的地名称: string
  - 目的地图片: image
  - 目的地介绍: text
  - 最佳旅游时间: string
  - 推荐理由: text
  - 相关线路: array[string]
- **热门目的地**: array[5-10]
  - 目的地信息同上
  - 热度指数: number

### 2.3 行程详情（模版）
- **行程信息**: object
  - 线路名称: string
  - 行程天数: number
  - 出发地: string
  - 目的地: string
  - 交通方式: string
  - 住宿标准: string
- **每日行程**: array
  - 第几天: number
  - 标题: string
  - 行程安排: text
  - 景点: array
    - 景点名称: string
    - 景点图片: image
    - 景点介绍: text
  - 住宿: string
  - 用餐: string
- **费用说明**: object
  - 费用包含: text
  - 费用不含: text
  - 儿童标准: text
- **预订须知**: text

### 2.4 特色服务
- **核心优势**: array[3-6]
  - 优势标题: string (如: 品质保证、专业导游、价格透明)
  - 优势描述: text
  - 优势图标: image
- **服务保障**: array
  - 保障项目: string (如: 签证办理、保险赠送)
  - 保障说明: text
- **导游团队**: array[5-10]
  - 导游姓名: string
  - 导游照片: image
  - 从业年限: number
  - 带团经验: text
  - 服务宣言: text
  - 好评率: number

### 2.5 景点门票
- **门票列表**: array[20-50]
  - 景点名称: string
  - 景点图片: image
  - 景点介绍: text
  - 门票价格: number
  - 开放时间: string
  - 景点地址: string
  - 预订链接: string
- **热门景点**: array[10-20]
  - 景点信息同上
  - 销量: number
  - 推荐指数: number

### 2.6 旅游攻略
- **攻略列表**: array[20-50]
  - 攻略标题: string
  - 目的地: string
  - 发布时间: date
  - 攻略摘要: text
  - 详细内容: text
  - 相关图片: array[image]
  - 攻略链接: string

### 2.7 用户游记
- **游记列表**: array[10-20]
  - 游记标题: string
  - 用户昵称: string
  - 目的地: string
  - 游记内容: text
  - 游记图片: array[image]
  - 发布时间: date
  - 点赞数: number

### 2.8 优惠活动
- **活动列表**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image
- **特价线路**: array[5-10]
  - 线路信息同上
  - 原价: number
  - 特价: number
  - 折扣: string

### 2.9 数据统计
- **业务数据**: object
  - 服务游客数: number
  - 旅游线路数: number
  - 覆盖目的地数: number
  - 导游人数: number

### 2.10 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
