# 时政信息行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **机构名称**: string - 时政信息机构全称
- **机构Logo**: image - 机构标志图片
- **行业类型**: string - "时政信息"
- **成立时间**: date - 机构成立日期
- **机构性质**: string - 如: 官方媒体、新闻平台

### 1.2 机构介绍
- **机构简介**: text - 200-500字机构介绍
- **办刊宗旨**: text - 办刊理念、使命
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 机构宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **新闻热线**: string
- **投稿邮箱**: string
- **机构地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、时政信息行业特性内容

### 2.1 新闻资讯
- **新闻分类**: array
  - 类别名称: string (如: 时政、经济、社会、国际、科技)
  - 新闻列表: array
    - 新闻标题: string
    - 新闻封面: image
    - 新闻摘要: text
    - 发布时间: date
    - 来源: string
    - 浏览量: number
    - 新闻详情链接: string
- **头条新闻**: array[5-10]
  - 新闻信息同上
  - 置顶标识: boolean

### 2.2 专题报道
- **专题列表**: array[10-20]
  - 专题标题: string
  - 专题封面: image
  - 专题介绍: text
  - 相关新闻: array[string]
  - 创建时间: date
  - 专题详情链接: string

### 2.3 政策解读
- **政策文件**: array[20-50]
  - 文件标题: string
  - 发布部门: string
  - 发布时间: date
  - 文件摘要: text
  - 详细内容: text
  - 文件链接: string
- **解读文章**: array[20-50]
  - 文章标题: string
  - 解读政策: string
  - 发布时间: date
  - 文章内容: text
  - 文章链接: string

### 2.4 深度报道
- **深度文章**: array[10-20]
  - 文章标题: string
  - 文章封面: image
  - 作者: string
  - 发布时间: date
  - 文章摘要: text
  - 详细内容: text
  - 相关图片: array[image]
  - 文章链接: string

### 2.5 视频新闻
- **视频列表**: array[10-20]
  - 视频标题: string
  - 视频封面: image
  - 视频时长: string
  - 发布时间: date
  - 播放量: number
  - 视频链接: string

### 2.6 评论专栏
- **评论文章**: array[10-20]
  - 文章标题: string
  - 评论员: string
  - 发布时间: date
  - 文章内容: text
  - 文章链接: string
- **热门评论**: array[10-20]
  - 评论内容: text
  - 点赞数: number
  - 评论时间: date

### 2.7 数据可视化
- **数据图表**: array[10-20]
  - 图表标题: string
  - 图表类型: string
  - 数据说明: text
  - 图表图片: image
  - 相关文章: string

### 2.8 数据统计
- **平台数据**: object
  - 日均发稿量: number
  - 注册用户数: number
  - 日活跃用户: number
  - 总阅读量: number

### 2.9 订阅服务
- **订阅方式**: array
  - 方式: string (如: RSS订阅、邮件订阅、APP推送)
  - 说明: text
- **个性化推荐**: object
  - 推荐机制: text
  - 设置方式: text
