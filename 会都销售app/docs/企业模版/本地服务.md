# 本地服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string
- **企业Logo**: image
- **行业类型**: string - "本地服务"
- **成立时间**: date

### 1.2 企业介绍
- **企业简介**: text
- **服务理念**: text

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]

### 1.4 联系方式
- **客服电话**: string
- **企业地址**: string
- **APP下载**: object

---

## 二、本地服务行业特性内容

### 2.1 服务分类
- **类别列表**: array
  - 类别名称: string (如: 家政、维修、搬家)
  - 服务列表: array
    - 服务名称: string
    - 服务介绍: text
    - 服务价格: string

### 2.2 服务商家
- **商家列表**: array[10-50]
  - 商家名称: string
  - 商家地址: string
  - 服务评分: number
  - 联系方式: string

### 2.3 数据统计
- **平台数据**: object
  - 商家数量: number
  - 服务订单数: number

---

## 三、页面结构建议

1. 企业介绍
2. 服务分类
3. 服务商家
4. 数据统计
5. 联系方式
