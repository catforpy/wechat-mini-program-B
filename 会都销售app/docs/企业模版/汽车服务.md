# 汽车服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 汽车服务企业全称
- **企业Logo**: image
- **行业类型**: string - "汽车服务"
- **成立时间**: date
- **服务规模**: string

### 1.2 企业介绍
- **企业简介**: text
- **服务理念**: text
- **发展历程**: array

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array

### 1.4 联系方式
- **客服电话**: string
- **救援电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、汽车服务行业特性内容

### 2.1 汽车品牌
- **品牌列表**: array[10-50]
  - 品牌名称: string
  - 品牌Logo: image
  - 品牌介绍: text
  - 车型列表: array
    - 车型名称: string
    - 车型图片: image
    - 车型价格: string
    - 车型详情链接: string

### 2.2 新车销售
- **热门车型**: array[10-20]
  - 车型名称: string
  - 车型图片: image
  - 指导价: number
  - 优惠价: number
  - 车型详情链接: string

### 2.3 二手车
- **二手车列表**: array[20-50]
  - 车型名称: string
  - 车辆图片: image
  - 上牌时间: date
  - 里程数: number
  - 价格: number
  - 车辆详情链接: string

### 2.4 售后服务
- **服务项目**: array
  - 项目名称: string (如: 保养、维修、美容)
  - 项目介绍: text
  - 项目价格: string

### 2.5 门店信息
- **门店列表**: array[10-50]
  - 门店名称: string
  - 门店地址: string
  - 营业时间: string
  - 联系电话: string

### 2.6 数据统计
- **业务数据**: object
  - 门店数量: number
  - 销售车辆数: number
  - 服务客户数: number

---

## 三、页面结构建议

1. 企业介绍

3. 新车销售
4. 二手车
5. 售后服务

7. 数据统计
