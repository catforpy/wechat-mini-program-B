# 深度合成行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - AI企业全称
- **企业Logo**: image
- **行业类型**: string - "深度合成"
- **成立时间**: date
- **技术规模**: string

### 1.2 企业介绍
- **企业简介**: text
- **技术理念**: text
- **发展历程**: array

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array

### 1.4 联系方式
- **商务合作**: string
- **技术支持**: string
- **企业地址**: string
- **官方网站**: string

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、深度合成行业特性内容

### 2.1 AI产品
- **产品分类**: array
  - 类别名称: string (如: 视频生成、图像生成、语音合成)
  - 产品列表: array
    - 产品名称: string
    - 产品介绍: text
    - 产品功能: text
    - 产品图片: image

### 2.2 技术能力
- **核心技术**: array[5-10]
  - 技术名称: string
  - 技术介绍: text
  - 技术图片: image

### 2.3 应用场景
- **场景列表**: array[10-20]
  - 场景名称: string
  - 场景介绍: text
  - 场景图片: image

### 2.4 API服务
- **API列表**: array
  - API名称: string
  - API介绍: text
  - API文档: string

### 2.5 数据统计
- **平台数据**: object
  - 注册用户数: number
  - 日处理量: number
  - API调用量: number

---
