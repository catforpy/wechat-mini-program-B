# 物流服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 企业/公司全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "物流服务"
- **成立时间**: date - 企业成立日期
- **企业规模**: string - 员工人数/车队规模

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **企业文化**: text - 企业使命、愿景、价值观
- **发展历程**: array - 历史里程碑列表
  - 时间: date
  - 事件: string
  - 图片: image (可选)

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频（MP4格式，时长3-5分钟）
- **宣传图片轮播**: array[5-10] - 企业风采图片
  - 图片: image
  - 标题: string
  - 描述: text
- **荣誉资质**: array - 资质证书图片
  - 证书名称: string
  - 证书图片: image
  - 获得时间: date

### 1.4 联系方式
- **客服电话**: string - 客服热线
- **投诉电话**: string - 投诉建议电话
- **企业地址**: string - 详细地址
- **营业时间**: string - 营业时间说明
- **官方网站**: string - 网址链接
- **微信公众号**: image - 公众号二维码

### 1.5 用户评价
- **用户评分**: number - 1-5星评分
- **评价总数**: number
- **用户好评**: array[10-20] - 用户评价列表
  - 用户头像: image
  - 用户昵称: string
  - 评分: number
  - 评价内容: text
  - 评价时间: date
  - 配图: array[image] (可选)

---

## 二、物流服务行业特性内容

### 2.1 运力展示
- **车队规模**: object
  - 车辆总数: number
  - 车型分类: array
    - 车型名称: string (如: 9.6米厢式货车、冷链车、危化品车)
    - 车辆数量: number
    - 车辆图片: image
  - 车辆图片展示: array[10-20] - 各类型车辆图片

### 2.2 服务范围
- **覆盖区域**: object
  - 全国网点数: number
  - 覆盖省市: array - 省份列表
  - 网点分布图: image - 地图展示
- **配送时效**: array
  - 线路: string (如: 北京-上海)
  - 时效: string (如: 次日达、48小时达)
  - 价格区间: string

### 2.3 物流团队
- **团队规模**: object
  - 司机人数: number
  - 配送员人数: number
  - 客服人数: number
- **司机团队展示**: array[5-10]
  - 司机姓名: string
  - 从业年限: number
  - 服务单数: number
  - 好评率: number
  - 个人照片: image
  - 荣誉: string (可选)
- **配送员团队展示**: array[5-10]
  - 配送员姓名: string
  - 服务区域: string
  - 日均配送: number
  - 服务评分: number
  - 个人照片: image

### 2.4 特色服务
- **核心优势**: array[3-6]
  - 优势标题: string (如: 24小时不间断服务)
  - 优势描述: text
  - 优势图标: image
- **增值服务**: array
  - 服务名称: string (如: 上门取件、代打包、冷链保鲜)
  - 服务说明: text
  - 收费标准: string
  - 服务图片: image (可选)
- **保价服务**: object
  - 是否支持: boolean
  - 保价费率: string
  - 理赔说明: text

### 2.5 智能化服务
- **手机APP**: object
  - APP下载链接: string
  - APP二维码: image
  - 功能介绍: array
    - 功能名称: string
    - 功能描述: text
    - 功能截图: image
- **微信小程序**: object
  - 小程序码: image
  - 功能介绍: array
- **实时追踪**: object
  - 是否支持: boolean
  - 追踪方式: string (如: GPS定位、扫码查询)
  - 演示视频: video (可选)

### 2.6 安心保障
- **安全保障**: array[3-6]
  - 保障项目: string (如: 货物保险、全程监控、专人押运)
  - 保障说明: text
  - 保障图片: image
- **服务承诺**: object
  - 准时送达承诺: text
  - 损坏赔偿承诺: text
  - 投诉处理承诺: text
  - 承诺书图片: image (可选)
- **保险服务**: object
  - 合作保险公司: array
    - 公司名称: string
    - 公司Logo: image
  - 保险种类: array
    - 保险名称: string
    - 保险范围: text
    - 保费说明: string

### 2.7 成功案例
- **客户案例**: array[5-10]
  - 客户名称: string
  - 客户Logo: image
  - 合作内容: text
  - 合作时长: string
  - 案例图片: array[image]
- **合作企业**: array[10-20] - 合作伙伴Logo展示
  - 企业名称: string
  - 企业Logo: image

### 2.8 价格体系
- **价格说明**: text - 计费方式说明
- **价格表**: array
  - 服务类型: string (如: 同城配送、省内配送、跨省配送)
  - 首重价格: number
  - 续重价格: number
  - 时效说明: string
- **优惠活动**: array
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image

### 2.9 常见问题
- **FAQ**: array[10-20]
  - 问题: string
  - 答案: text
  - 展开图标: image (可选)

## 三、页面结构建议

### 3.1 页面顶部
- 轮播宣传图/视频
- 企业Logo + 名称
- 核心数据展示（如: 日均配送xx万单、覆盖xx个城市）

### 3.2 快捷入口
- 一键下单
- 运费查询
- 订单追踪
- 在线客服

### 3.3 内容模块（按顺序）
1. 企业介绍
2. 运力展示（车辆、设备）
3. 服务范围 + 时效
4. 物流团队（司机、配送员）
5. 特色服务
6. 智能化服务（APP、小程序）
7. 安心保障
8. 成功案例
9. 价格体系
10. 用户评价
11. 常见问题
12. 联系方式

### 3.4 底部
- 关注公众号
- APP下载
- 客服电话
- 企业地址
