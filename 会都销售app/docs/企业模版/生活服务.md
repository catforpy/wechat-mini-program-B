# 生活服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 生活服务企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "生活服务"
- **成立时间**: date - 企业成立日期
- **服务规模**: string - 如: 覆盖xx个城市、商家xx万家

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **服务理念**: text - 服务宗旨、使命、愿景
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

## 二、生活服务行业特性内容

### 2.1 服务分类
- **服务类别**: array
  - 类别名称: string (如: 餐饮美食、休闲娱乐、美容美发、家政服务)
  - 服务列表: array
    - 服务名称: string
    - 服务简介: text
    - 服务价格: string
    - 服务图片: image
    - 服务详情链接: string
- **热门服务**: array[10-20]
  - 服务名称: string
  - 服务封面: image
  - 服务介绍: text
  - 销量: number
  - 好评率: number

### 2.2 商家入驻
- **商家列表**: array[10-50]
  - 商家名称: string
  - 商家Logo: image
  - 商家类型: string
  - 商家位置: string
  - 营业时间: string
  - 人均消费: string
  - 评分: number
  - 商家详情链接: string
- **推荐商家**: array[5-10]
  - 商家信息同上
  - 推荐理由: text

### 2.3 服务项目
- **项目分类**: array
  - 项目类别: string
  - 项目列表: array
    - 项目名称: string
    - 项目介绍: text
    - 项目时长: string
    - 项目价格: number
    - 项目图片: image
- **优惠套餐**: array[5-10]
  - 套餐名称: string
  - 套餐内容: text
  - 原价: number
  - 优惠价: number
  - 套餐图片: image

### 2.4 服务团队
- **团队概况**: object
  - 服务人员总数: number
  - 认证服务人员: number
- **服务团队**: array[5-10]
  - 服务人员姓名: string
  - 服务项目: string
  - 从业年限: number
  - 服务单数: number
  - 好评率: number
  - 个人照片: image
  - 个人简介: text

### 2.5 服务保障
- **核心优势**: array[3-6]
  - 优势标题: string (如: 专业服务、价格透明、售后保障)
  - 优势描述: text
  - 优势图标: image
- **服务承诺**: array
  - 承诺项目: string (如: 准时服务、不满意重做)
  - 承诺说明: text
  - 承诺图片: image
- **售后保障**: object
  - 保障期限: string
  - 保障范围: text
  - 保障说明: text

### 2.6 预订服务
- **在线预订**: object
  - 预订方式: array
    - 方式: string (如: APP预订、微信预订、电话预订)
    - 说明: text
  - 预订流程: text
  - 预订须知: text
- **优惠活动**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image

### 2.7 用户晒单
- **用户晒单**: array[10-20]
  - 用户昵称: string
  - 服务项目: string
  - 晒单图片: array[image]
  - 晒单内容: text
  - 晒单时间: date
  - 点赞数: number

### 2.8 数据统计
- **业务数据**: object
  - 服务客户数: number
  - 入驻商家数: number
  - 服务订单数: number
  - 覆盖城市数: number

### 2.9 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
