# 电商平台行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 电商平台全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "电商平台"
- **成立时间**: date - 企业成立日期
- **平台规模**: string - 如: 商家xx万、商品xx亿

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **平台理念**: text - 平台理念、使命
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 企业宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 资质证书

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、电商平台行业特性内容

### 2.1 商品分类
- **商品类别**: array
  - 类别名称: string (如: 服装鞋帽、家电数码、食品生鲜)
  - 子类别: array
    - 子类别名称: string
    - 商品列表: array
      - 商品名称: string
      - 商品图片: image
      - 商品价格: number
      - 销量: number
      - 商品详情链接: string
- **热门商品**: array[10-20]
  - 商品信息同上
  - 热度指数: number

### 2.2 店铺入驻
- **店铺列表**: array[10-50]
  - 店铺名称: string
  - 店铺Logo: image
  - 店铺类型: string
  - 主营商品: string
  - 店铺评分: number
  - 店铺详情链接: string
- **旗舰店**: array[5-10]
  - 店铺信息同上
  - 推荐理由: text

### 2.3 优惠活动
- **活动列表**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image
- **优惠券**: array[10-20]
  - 优惠券名称: string
  - 优惠券面额: string
  - 使用条件: text
  - 有效期: string
  - 领取链接: string

### 2.4 购物保障
- **核心优势**: array[3-6]
  - 优势标题: string (如: 正品保障、极速退款、售后服务)
  - 优势描述: text
  - 优势图标: image
- **服务承诺**: array
  - 承诺项目: string
  - 承诺说明: text
  - 承诺图片: image

### 2.5 物流服务
- **配送方式**: array
  - 方式: string (如: 快递配送、门店自提、同城配送)
  - 说明: text
  - 时效: string
  - 运费: string
- **物流信息**: object
  - 配送范围: text
  - 配送时效: text
  - 运费说明: text

### 2.6 售后服务
- **售后政策**: object
  - 退换货政策: text
  - 退款说明: text
  - 售后流程: text
- **客服服务**: array
  - 服务类型: string (如: 在线客服、电话客服)
  - 服务时间: string
  - 服务说明: text

### 2.7 数据统计
- **平台数据**: object
  - 注册用户数: number
  - 入驻商家数: number
  - 商品总数: number
  - 日订单数: number
