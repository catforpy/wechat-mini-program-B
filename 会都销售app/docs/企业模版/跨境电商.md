# 跨境电商行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string
- **企业Logo**: image
- **行业类型**: string - "跨境电商"
- **成立时间**: date

### 1.2 企业介绍
- **企业简介**: text
- **平台理念**: text

### 1.3 宣传内容
- **宣传片**: video
- **宣传图片轮播**: array[5-10]

### 1.4 联系方式
- **客服电话**: string
- **企业地址**: string
- **官方网站**: string
- **APP下载**: object

---

## 二、跨境电商行业特性内容

### 2.1 进口商品
- **商品分类**: array
  - 类别名称: string (如: 母婴、美妆、保健)
  - 商品列表: array
    - 商品名称: string
    - 商品图片: image
    - 原产地: string
    - 商品价格: number

### 2.2 购物指南
- **指南列表**: array
  - 指南标题: string
  - 指南内容: text

### 2.3 物流服务
- **物流信息**: object
  - 配送方式: array
  - 时效说明: text
  - 关税说明: text

### 2.4 数据统计
- **平台数据**: object
  - 商品数量: number
  - 用户数量: number

---

## 三、页面结构建议

1. 企业介绍
2. 进口商品
3. 购物指南
4. 物流服务
5. 数据统计
6. 联系方式
