# 金融业行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 金融机构全称
- **企业Logo**: image - 机构标志图片
- **行业类型**: string - "金融业"
- **成立时间**: date - 机构成立日期
- **机构性质**: string - 如: 银行、保险、证券、基金、支付平台

### 1.2 企业介绍
- **机构简介**: text - 200-500字机构介绍
- **服务理念**: text - 服务宗旨、使命、愿景
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 机构宣传视频
- **宣传图片轮播**: array[5-10]
- **荣誉资质**: array - 金融牌照、资质证书

### 1.4 联系方式
- **客服电话**: string - 客服热线
- **投诉电话**: string - 投诉建议电话
- **机构地址**: string
- **官方网站**: string
- **APP下载**: object
  - iOS链接: string
  - Android链接: string
  - 下载二维码: image

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

## 二、金融业行业特性内容

### 2.1 产品服务
- **产品分类**: array
  - 产品类别: string (如: 储蓄、贷款、理财、保险、基金)
  - 产品列表: array
    - 产品名称: string
    - 产品简介: text
    - 产品特点: text
    - 收益率: string
    - 风险等级: string
    - 起购金额: number
    - 产品期限: string
    - 产品图标: image
- **热门产品**: array[5-10]
  - 产品名称: string
  - 产品封面: image
  - 产品介绍: text
  - 收益率: string
  - 投资人数: number
  - 产品详情链接: string

### 2.2 核心优势
- **核心优势**: array[3-6]
  - 优势标题: string (如: 安全保障、收益稳健、操作便捷)
  - 优势描述: text
  - 优势图标: image
- **安全保障**: array
  - 保障项目: string (如: 资金托管、风控体系、保险保障)
  - 保障说明: text
  - 保障图片: image
- **技术优势**: array
  - 技术名称: string (如: 大数据风控、AI智能投顾)
  - 技术介绍: text

### 2.3 专业团队
- **团队概况**: object
  - 团队规模: number
  - 金融从业经验: string
  - 资质认证: array
- **专家团队**: array[5-10]
  - 专家姓名: string
  - 职位: string
  - 专业领域: string
  - 从业年限: number
  - 个人简介: text
  - 个人照片: image
- **顾问服务**: object
  - 服务介绍: text
  - 服务方式: array
    - 方式: string
    - 说明: text

### 2.4 数据统计
- **业务数据**: object
  - 用户总数: number
  - 交易总额: number
  - 理财总额: number
  - 累计收益: number
- **实时数据**: array
  - 数据名称: string
  - 数据值: number
  - 更新时间: date

### 2.5 优惠活动
- **活动列表**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image
  - 参与方式: string
- **新手福利**: object
  - 福利内容: text
  - 领取方式: string
  - 活动规则: text

### 2.6 理财知识
- **理财资讯**: array[20-50]
  - 标题: string
  - 资讯类别: string
  - 发布时间: date
  - 内容摘要: text
  - 详细内容: text
- **理财课堂**: array[10-20]
  - 课程标题: string
  - 课程介绍: text
  - 课程视频: video
  - 讲师: string
- **理财工具**: array
  - 工具名称: string
  - 工具介绍: text
  - 工具链接: string

### 2.7 安全中心
- **安全措施**: array[5-10]
  - 措施名称: string
  - 措施说明: text
  - 措施图片: image
- **安全提示**: array
  - 提示内容: text
- **风控体系**: object
  - 体系介绍: text
  - 体系图示: image

### 2.8 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
