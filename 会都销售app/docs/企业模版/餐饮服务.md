# 餐饮服务行业模版

## 一、共性内容（所有行业通用）

### 1.1 基础信息
- **企业名称**: string - 餐饮企业全称
- **企业Logo**: image - 企业标志图片
- **行业类型**: string - "餐饮服务"
- **成立时间**: date - 企业成立日期
- **门店数量**: string - 如: 全国xx家门店

### 1.2 企业介绍
- **企业简介**: text - 200-500字企业介绍
- **品牌理念**: text - 品牌文化、理念
- **发展历程**: array - 历史里程碑列表

### 1.3 宣传内容
- **宣传片**: video - 品牌宣传视频
- **宣传图片轮播**: array[5-10] - 菜品/门店环境图片
- **荣誉资质**: array - 资质证书、荣誉奖项

### 1.4 联系方式
- **客服电话**: string
- **投诉电话**: string
- **总部地址**: string
- **官方网站**: string
- **APP下载**: object

### 1.5 用户评价
- **用户评分**: number
- **评价总数**: number
- **用户好评**: array[10-20]

---

## 二、餐饮服务行业特性内容

### 2.1 菜品展示
- **菜品分类**: array
  - 类别名称: string (如: 热菜、凉菜、汤品、主食、饮品)
  - 菜品列表: array
    - 菜品名称: string
    - 菜品图片: image
    - 菜品介绍: text
    - 菜品价格: number
    - 销量: number
    - 口味标签: array (如: 微辣、清淡)
    - 菜品详情链接: string
- **招牌菜品**: array[5-10]
  - 菜品名称: string
  - 菜品图片: image
  - 菜品介绍: text
  - 价格: number
  - 销量: number
  - 好评率: number

### 2.2 门店信息
- **门店列表**: array[10-100]
  - 门店名称: string
  - 门店图片: array[3-5]
  - 门店地址: string
  - 营业时间: string
  - 联系电话: string
  - 人均消费: string
  - 评分: number
  - 门店详情链接: string
- **特色门店**: array[5-10]
  - 门店信息同上
  - 特色介绍: text

### 2.3 外卖服务
- **外卖菜品**: array[20-50]
  - 菜品信息同上
  - 外卖包装: text
  - 配送费: number
  - 配送时间: string
- **配送范围**: object
  - 配送区域: array[string]
  - 配送地图: image
- **配送服务**: array
  - 服务项目: string (如: 30分钟送达、准时必达)
  - 服务说明: text

### 2.4 优惠活动
- **活动列表**: array[5-10]
  - 活动标题: string
  - 活动内容: text
  - 活动时间: string
  - 活动海报: image
  - 使用规则: text
- **优惠券**: array[10-20]
  - 优惠券名称: string
  - 优惠券面额: string
  - 使用条件: text
  - 有效期: string
  - 领取链接: string

### 2.5 品牌故事
- **品牌介绍**: text - 品牌创立故事
- **品牌文化**: text - 品牌理念、价值观
- **创始人**: object (可选)
  - 创始人姓名: string
  - 创始人照片: image
  - 创始人介绍: text

### 2.6 环境展示
- **门店环境**: array[20-50]
  - 图片类型: string (如: 大厅、包间、外观)
  - 图片: image
  - 图片说明: string
- **设施服务**: array
  - 设施名称: string (如: 包间、停车场、WiFi)
  - 设施说明: text
  - 设施图片: image

### 2.7 用户晒单
- **用户晒单**: array[10-20]
  - 用户昵称: string
  - 晒单图片: array[image]
  - 晒单内容: text
  - 晒单时间: date
  - 点赞数: number
- **美食点评**: array[10-20]
  - 点评内容: text
  - 点评图片: array[image]
  - 点评时间: date

### 2.8 数据统计
- **业务数据**: object
  - 门店数量: number
  - 服务顾客数: number
  - 日均订单数: number
  - 覆盖城市数: number

### 2.9 常见问题
- **FAQ**: array[20-50]
  - 问题分类: string
  - 问题: string
  - 答案: text
