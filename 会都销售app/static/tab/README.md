# Tab 图标说明

此目录用于存放底部导航栏图标。

需要以下图标文件（每个48x48px）：

## 首页图标
- home.png - 未选中状态
- home-active.png - 选中状态

## 视频图标
- video.png - 未选中状态
- video-active.png - 选中状态

## 消息图标
- message.png - 未选中状态
- message-active.png - 选中状态

## 直播图标
- live.png - 未选中状态
- live-active.png - 选中状态

## 我的图标
- profile.png - 未选中状态
- profile-active.png - 选中状态

可以使用以下方式创建图标：
1. 使用iconfont Unicode字符
2. 使用在线图标库下载
3. 自己设计图标

暂时可以先使用纯文字模式，将pages.json中的iconPath和selectedIconPath暂时注释掉。
